
## 1. Product Overview
ERP管理系统，用于企业资源管理，包括用户管理、菜单管理、产品管理和权限控制功能。
- 主要目标是提供一个灵活的企业管理平台，通过用户-用户组-菜单的权限模型，实现不同用户看到不同界面的效果。
- 为中小企业提供轻量级、易于部署和管理的ERP解决方案。

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| 管理员 | 系统初始化创建 | 所有功能权限，包括用户管理、菜单管理、产品管理、用户组管理 |
| 普通用户 | 管理员创建或自行注册 | 根据所属用户组的权限，访问相应的菜单和功能 |

### 2.2 Feature Module
1. **登录页**: 用户登录、JWT Token认证
2. **主界面布局**: 侧边栏菜单、顶部导航、内容区域
3. **用户管理**: 用户列表、新增、编辑、禁用、删除
4. **用户组管理**: 用户组列表、新增、编辑、删除、分配菜单权限
5. **菜单管理**: 菜单列表、新增、编辑、删除、排序、隐藏/显示
6. **产品管理**: 产品列表、新增、编辑、删除、搜索、分页

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 登录页 | 登录表单 | 用户名密码登录，记住登录状态，JWT Token管理 |
| 用户管理 | 用户列表 | 分页显示用户，支持搜索，显示用户状态（启用/禁用） |
| 用户管理 | 用户编辑 | 新增/编辑用户信息，分配用户组，修改密码 |
| 用户组管理 | 用户组列表 | 显示所有用户组，管理用户组权限 |
| 用户组管理 | 权限分配 | 树形选择菜单权限，保存用户组权限配置 |
| 菜单管理 | 菜单列表 | 树形结构显示菜单，支持拖拽排序 |
| 菜单管理 | 菜单编辑 | 新增/编辑菜单，设置父级菜单、图标、路径、排序 |
| 产品管理 | 产品列表 | 分页显示产品，按名称搜索，显示产品基础信息 |
| 产品管理 | 产品编辑 | 新增/编辑产品，填写产品名称、编码、价格、规格等信息 |

## 3. Core Process
### 用户登录流程
用户输入账号密码 → 后端验证 → 生成JWT Token → 返回用户信息和菜单权限 → 前端渲染界面

### 权限控制流程
用户登录 → 获取用户所属用户组 → 获取用户组关联的菜单 → 前端动态渲染侧边栏 → 后端API验证权限

### 菜单管理流程
管理员登录 → 菜单管理页面 → 新增/编辑/删除菜单 → 保存到数据库 → 前端刷新菜单

```mermaid

    A[用户登录] --&gt; B{验证账号密码}
    B --&gt;|成功| C[生成JWT Token]
    C --&gt; D[获取用户组信息]
    D --&gt; E[获取用户组菜单权限]
    E --&gt; F[返回用户信息和菜单]
    F --&gt; G[前端渲染界面]
    B --&gt;|失败| H[返回错误提示]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：蓝色系 (#409EFF)，体现企业级系统的专业性
- 辅助色：灰色系 (#909399)，用于次要信息
- 按钮风格：圆角矩形，Element Plus默认风格
- 字体：14px 微软雅黑/苹方，清晰易读
- 布局风格：左右分栏，左侧固定侧边栏，右侧内容区域
- 图标风格：Element Plus 内置图标

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 登录页 | 登录表单 | 居中卡片布局，用户名、密码输入框，登录按钮 |
| 主界面 | 侧边栏 | 蓝色背景，白色文字，支持折叠，菜单高亮 |
| 主界面 | 顶部导航 | 面包屑导航，用户信息，退出登录 |
| 管理页面 | 数据表格 | 斑马纹，分页器，操作列（编辑、删除） |
| 管理页面 | 弹窗表单 | 居中弹窗，表单验证，保存/取消按钮 |

### 4.3 Responsiveness
- 桌面端优先设计
- 侧边栏可折叠
- 表格支持横向滚动
- 弹窗自适应屏幕大小
